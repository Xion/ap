Succinct & readable processing language for a Unix shell. Written in Rust.

    $ echo "Hello " | rh '_ + world'
    Hello world

    $ echo '1,2,3' | rh 'split(",") & map(int & (*2)) & join(", ")'
    2, 4, 6
